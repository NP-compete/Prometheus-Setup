# Prometheus-Setup

## Prerequisites
1. Linux 
2. Docker


## To run prometheus

1. git clone https://github.com/NP-compete/Prometheus-Setup
2. sudo sh start.sh

## To get docker metrics

1. git clone https://github.com/NP-compete/Prometheus-Setup
2. sudo sh docker-metrics.sh
3. sudo sh start.sh
